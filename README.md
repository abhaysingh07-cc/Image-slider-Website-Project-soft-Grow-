# Image-slider-Website-Project-soft-Grow-

Advanced Image Slider -
A professional, high-performance Image Slider (Carousel) built with a focus on modern UI/UX principles. This project features a clean "SkyBlue" theme, fully responsive design, and advanced JavaScript logic for both desktop and mobile users.

✨ Key Features
📱 Advanced Touch Support: Fully optimized for mobile devices. Users can swipe left or right on the screen to navigate between images.

⏱️ Intelligent Auto-Play: Images transition automatically every 6 seconds. The timer intelligently resets whenever a user interacts with the slider (clicks or swipes).

🎨 SkyBlue Aesthetic: A modern design featuring a skyblue background, glassmorphism captions (blurred backgrounds), and smooth fade transitions.

🔗 Interactive Navigation: Includes "Next" and "Previous" buttons, along with clickable pagination dots that show the current slide status.

📐 Fully Responsive: Adapts seamlessly to all screen sizes, from large desktop monitors to small mobile phones.

🛠️ Tech Stack
HTML5: Semantic structure for accessibility.

CSS3: Flexbox, smooth animations, and a custom SkyBlue theme.

JavaScript (ES6): Custom touch-tracking logic, event listeners, and DOM manipulation.

🚀 Quick Start
Download: Save index.html, style.css, and script.js into the same folder.

Run: Open index.html in any modern web browser.

Interact: Click the arrows, use keyboard arrow keys, or swipe if you are on a mobile device.

📝 Customization
Change Images: Update the src attributes in the .slide img tags in index.html.

Adjust Speed: Change the intervalTime variable in script.js (currently set to 6000 for 6 seconds).

Theme: Modify the background-color in style.css under the body selector to change the skyblue theme.
